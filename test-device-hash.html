Current Device Hash

How It Works

Test Procedure:
  1. Click "Generate New Hash" - ini akan buat device fingerprint
  2. Hash disimpan di localStorage (persistent)
  3. Click "Compare with Stored" untuk cek apakah hash sama
  4. Refresh halaman dan compare lagi - seharusnya SAMA
  5. Clear browser data (localStorage) - hash akan berbeda
